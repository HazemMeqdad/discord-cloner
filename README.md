## ‚ú® Commands
- restart:
  - `.restart`: reset server

- copy:
    - `.copy -all`: copy all options from server
    - `.copy -roles`: copy roles
    - `.copy -emojis`: copy emojis
    - `.copy -settings`: copy settings example: icon, name, description, ...

- exit:
    - `.exit`: close the tool
